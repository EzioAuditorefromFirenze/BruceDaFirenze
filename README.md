# Arcane Insights - Tarot Reading Website

A beautifully designed, fully functional tarot reading website with authentic readings, card animations, and user behavior tracking via cookies.

## Features

✨ **Authentic Tarot Experience**
- Pre-reading questions (name, question, life area, spread choice)
- 42 real tarot cards (Major & Minor Arcana) with authentic meanings
- Reversed cards for deeper interpretation
- Context-aware readings based on selected life area

🎨 **Stunning Design**
- Glassmorphic UI with backdrop blur effects
- Smooth 3D card flip animations
- Gradient overlays and atmospheric lighting
- Fully responsive (mobile, tablet, desktop)

🍪 **Smart Tracking**
- Saves reading history (last 20 readings)
- Auto-fills visitor name on return
- Remembers preferred life categories
- Non-intrusive cookie-based tracking

2. Set source to `main` branch

## How It Works

1. **Enter Your Details** - Name, question, life area, and spread type
2. **Reveal Your Cards** - Click to flip and discover each card
3. **Read The Message** - Get an AI-generated interpretation
4. **Track History** - All readings saved in your browser cookies

## Technology Stack

- Pure HTML5 (no build tools needed)
- Vanilla CSS3 (animations, gradients, 3D transforms)
- Vanilla JavaScript (zero dependencies)
- Browser Cookies API (user tracking)
